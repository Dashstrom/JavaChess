# JavaChess

Projet de jeu d'échec en console fait en Java orienté fonctionnel.
Pour jouer c'est tout simple, lancer la class Game !  

## Rêgles

Toute les règles si-dessous sont implémenter :

- rock
- prise en passant
- echec/echec et mat
- manque materiel
- triple répétition
- règle des 50 et 75 coups
- promotions
- pat
